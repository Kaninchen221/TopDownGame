## TopDownGame
### Introduction
Open source small TopDown 2D game in UE4  
Target is 15m - 30m of gameplay  
I am currently creating gameplay mechanics  

### Technology
+ Unreal Engine 4.26
+ Paper2D module
+ C++/Blueprints
+ Git
